Instruções para atualizar um código HTML com novas imagens e links:

Estrutura atual do código HTML:
O código HTML atual contém duas seções <div class="external-links-container">, cada uma contendo uma série de links de imagens.
Cada link de imagem é um elemento <a> com um atributo href que corresponde a uma função openLightbox() e um elemento <img> dentro, com um atributo src que define o caminho da imagem e um atributo alt para texto alternativo.
Localização das novas imagens:
As novas imagens estão localizadas no diretório "IMG/".
Configuração dos novos links:
Os novos links devem abrir uma lightbox ao clicar, assim como os links existentes.
Os atributos href dos links devem ser configurados para chamar a função openLightbox() com o caminho completo da imagem como argumento.
Os atributos src das imagens devem ser configurados com os caminhos completos das novas imagens.
Diretórios relevantes onde as imagens estão localizadas:
As imagens estão localizadas no diretório "IMG/".
Passo a passo para substituir ou adicionar os links e imagens no código HTML:
Para cada nova imagem:
Adicione um novo elemento <a> dentro da seção <div class="external-links-container">.
Configure o atributo href para chamar a função openLightbox() com o caminho completo da imagem como argumento (por exemplo, onclick="openLightbox('IMG/nome_da_imagem.jpg')".
Adicione um elemento <img> dentro do elemento <a>.
Configure o atributo src do elemento <img> com o caminho completo da nova imagem (por exemplo, src="IMG/nome_da_imagem.jpg").
Forneça um texto alternativo apropriado usando o atributo alt.
